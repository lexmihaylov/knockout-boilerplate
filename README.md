A component based app boilerplate using knockout, requirejs and less.

# Download

```bash
mkdir my-project
cd my-project
git clone https://github.com/lexmihaylov/knockout-boilerplate.git ./
```

# Setup

```bash
npm install
bower install
```

# Building
```bash
npm run build
```
